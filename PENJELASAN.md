Program ini digunakan untuk menghitung total biaya pengiriman paket berdasarkan berat, jarak, dan ukuran paket
Deskripsi Program : 
- Berat paket (kg)
- Jarak (m)
- Ukuran paket (panjang, lebar, dan tinggi)

- Jika volume paket > 100, dikenakan tambahan biaya sebesar Rp 50.000

Cara Kerja Program : 
1. Pengguna memasukkan berat, jarak, panjang, lebar, dan tinggi paket
2. Program menentukan biaya berdasarkan jarak
3. Program menghitung volume paket
4. Jika volume lebih dari 100, tambahan biaya 50.000
5. Hasil total biaya dilampirkan di terminal
